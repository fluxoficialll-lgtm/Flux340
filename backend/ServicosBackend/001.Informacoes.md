# Documentação da Camada de Serviços (Backend)

O diretório `/backend/ServicosBackend` contém a lógica de negócio central da aplicação. Esta camada é responsável por orquestrar as operações, validar permissões e processar os dados recebidos da camada de Rotas antes de interagir com as camadas inferiores (como a de acesso a dados).

## Estrutura e Organização

A nomenclatura dos arquivos de serviço espelha a das rotas, mantendo a consistência e a clareza:

`Servicos.{Dominio}.{Subdominio}.js`

### Principais Responsabilidades:

- **Lógica de Negócio:** Implementa as regras e os fluxos de trabalho da aplicação. Por exemplo, no `Servicos.Criacao.Grupo.Pago.js`, a lógica para criar um grupo, associar um preço e configurar permissões é executada.

- **Orquestração:** Coordena as chamadas para outras camadas. Um serviço pode chamar um ou mais repositórios ou outras camadas de acesso a dados para cumprir uma tarefa. Por exemplo, ao criar um post (`Servicos.Publicacao.Feed.js`), o serviço pode precisar verificar os dados do usuário e depois solicitar a inserção do post no banco de dados.

- **Validação e Segurança:** Realiza validações de dados mais complexas e verifica as permissões do usuário. Por exemplo, um serviço pode verificar se o usuário que tenta deletar um post é de fato o autor do post.

- **Serviços de Sistema:**
  - `Servico.Logs.Backend.js`: Centraliza a lógica de logging para auditoria e depuração.
  - `Sistema.Taxas.Stripe.js`: Encapsula a lógica para cálculo de taxas de transação específicas do Stripe.

## Fluxo de Dados

1.  A camada de `Rotas` recebe uma requisição HTTP e extrai os dados relevantes (corpo da requisição, parâmetros, etc.).
2.  A rota chama a função de serviço correspondente, passando esses dados.
3.  O serviço executa a lógica de negócio, interage com o banco de dados (através da camada de `Consultas` ou `Repositorios`) e realiza as operações necessárias.
4.  O serviço retorna o resultado (ou um erro) para a camada de `Rotas`.
5.  A rota envia a resposta HTTP final de volta para o cliente.
